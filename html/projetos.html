Viajar é Preciso

See the Pen Viajar é preciso by Jessica Ferraz (@ferrazjaa) on CodePen.

Este projeto consiste em uma página inicial para uma agência de viagens fictícia, desenvolvida com o uso do Bootstrap 5. Destacam-se três principais recursos: um modal de "Fale Conosco" com efeito fade para facilitar o contato com a agência, um tema Dark Mode que permite aos usuários escolher entre os modos claro e escuro para uma experiência visual personalizada, e um carrossel automático que exibe destinos de viagem de forma atrativa.
